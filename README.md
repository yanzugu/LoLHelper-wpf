# LoLHelper-wpf

## 英雄聯盟小幫手
### 功能 :
- 自動接受
- 自動選角
- 自動喊路
- 自動更換符文(From OPGG)
- 顯示隊友牌位(於選角聊天室)
- 自動列隊

### 使用方法 :
1. 啟動遊戲。
1. 於最上方的 input box 輸入 LeagueClient 資料夾路徑後按確定。
1. 勾選功能後按下開始按鈕。
